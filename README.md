# Manage My Shifts

Manage My Shifts is a client-side shift management web application developed as part of an academic JavaScript project.

The application provides a structured multi-page workflow for managing user and shift-related information, including authentication, profile updates, resume updates, shift viewing, and add/update shift actions.

## Live Demo
[View the live project](https://afikyermi.github.io/manage-my-shifts/)

## Purpose
The purpose of this project was to build a full front-end application from scratch and practice page navigation, data handling, validation logic, and user experience flow in a structured academic environment.

## Project Overview
This project was built to demonstrate practical front-end development skills using HTML, CSS, and JavaScript only.

It simulates a real-world shift management workflow and emphasizes:
- clear page structure
- modular front-end files
- form interaction and validation
- local client-side data storage
- maintainable and readable code

## Main Features
- User login interface
- User registration flow
- Forgot password screen
- Profile update functionality
- Resume update functionality
- Shift management screen
- Add / update shift workflow
- Client-side data persistence using LocalStorage

## Technologies Used
- HTML5
- CSS3
- JavaScript
- LocalStorage
- GitHub Pages

## Project Structure
- Separate HTML pages for each main workflow
- Dedicated CSS files for page styling
- Dedicated JavaScript files for page behavior and logic
- Root `index.html` entry point for deployment

## Notes
This project is fully client-side and does not include server-side logic or database integration.
